Hướng dẫn chạy file code
1) Sau khi tải file code về tạo một thư mục trống, chuyển file code đã tải về vào thư mục vừa tạo
2) Mở màn hình command prompt hoặc terminal trên vs code, chuyển hướng đến thư mục .venv và gõ Scripts/activate đối với window để tạo môi trường ảo
3) Tiếp tục chuyển hướng đến thư mục facereg sau đó gõ py manage.py runserver để chạy chương trình, nó sẽ xuất hiện địa chỉ của trang web
